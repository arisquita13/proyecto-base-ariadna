# Proyecto Base - Actividad 2 (Ari)

Este repo es la base para la práctica colaborativa de Git y GitHub.

## Lo que ya incluye (anfitrión)
- Estructura del proyecto
- `main.py` con ejemplo de uso
- Clase `Avanzadas` con: __init__(), leerNumeros(), elevarPotencia(), mostrarResultado()

## Tu tarea (colaborador)
1) Crear una rama con tu nombre (ej. `dev-tu-nombre`).
2) Implementar la operación **raizCuadrada()** en `clases/avanzadas.py`.
   - Lee un número (puedes reutilizar leerNumeros o crear entrada simple).
   - Calcula la raíz cuadrada (math.sqrt o x ** 0.5).
   - Muestra el resultado en `mostrarResultado()` o imprime desde main.
3) Agregar en `main.py` una opción simple para ejecutar `raizCuadrada()` (menú o llamada directa).
4) Hacer commit y push de tu rama.
5) Crear un Pull Request (de tu rama hacia `master` de tu fork) y notificar para revisión.

## Evidencias que debes generar (según práctica)
- Fork, commit, push, PR y merge (cada quien según su rol).
